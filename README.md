# Employee CRUD (Node + React + PostgreSQL)

## 📦 Setup Instructions

### 1. Create Database

Open PostgreSQL / pgAdmin and create database:

```
employeecrud
```

---

### 2. Import SQL File

* Open Query Tool
* Open `backend/database.sql`
* Click Run ▶

---

### 3. Backend Setup

```
cd backend
npm install
node server.js
```

---

### 4. Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

### 5. Open App

```
http://localhost:5173
```

---

## 🧠 Tech Stack

* Node.js + Express
* PostgreSQL
* React (Vite)
* Axios
